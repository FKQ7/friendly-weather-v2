Introduction to the project

Friendly Weather is a Forecasting prediction model that uses data to predict the weather, simple right? it won't be in the future, the idea of an assistant for a meteorologist that can help him/her and reduce the misleading info about the weather, how reliable is forecasting information? in 10 days the first 5 are 90% accurate and the next 2 are 80% but the days 8,9,10 are 50%! what if we could save lives by gathering thousands of data in seconds and predicting it more accurately? we can't stop the tsunami but we can get a heads-up earlier


How did we build it?

We utilized Python and Machine learning to make our Web application come to life, we fetched data to a model called "Weather Predicter" that used those data to predict the weather, we took weather data from 4 different sources, sources will be mentioned later, and we added an AI tool for weather collection to convert the data to a CSV file so you can make data easily
